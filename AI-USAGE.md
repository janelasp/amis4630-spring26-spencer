# AI Usage (Copilot)

## Component Hierarchy
AI was used to help with construction of component hierarchy and diagram as well as refine architecture in relation to pain points from user stories.

## Product Catalog Development
I asked AI on how I should go about making the product catalog and where exactly should I start. It suggested starting from the backend and then focusing on the frontend. I also asked about different terminologies that I wasn't familiar with such as using 'm' for decimal numbers. I ignored the AI suggestions for the types of products I wanted to display on the application. There would also be instances where it told me to import certain stuff such as BrowserRouter but I chose not to because I didn't see that in any of the materials we learned and observed in lecture. I also asked it to explain certain stuff such as onViewDetails, onBack and IEnumerable.

## Shopping Cart Development
I used AI to check if I met the requirements for the Milestone's deliverables. I also asked it to explain the concepts of payload and useReducer so I could better understand it. AI was also used to assist with fixing issues such as the product catalog not displaying on the application and integrating the frontend and backend.


## Authentication, Security & Testing Development
I used AI to create the testing and security agents as well generate tests for the frontend and backend. It was also used to help with implementing user authentication with JWT, protection of API endpoints as well as building the checkout/order flow. Copilot was also used to correct errors with a minimal fix. It was also prompted to not generate exhaustive lines of code and lose its track.
