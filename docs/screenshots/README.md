# Screenshots for User/Admin Guides

Put guide screenshots in this folder and keep filenames stable so the markdown docs render correctly.

## Recommended capture settings

- Desktop screenshots: ~1280×720 or similar.
- Mobile screenshots (optional): use browser devtools device emulation (iPhone 13 / Pixel 5).
- Keep the browser zoom at 100%.
- Prefer light mode and a clean seed database state.
