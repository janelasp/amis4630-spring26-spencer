## 1) Manage Products

**Goal:** Create, update, and delete products.

### Create a product
1. Navigate to the Admin section.
2. Go to Product Management section
3. Fill in the product fields (title, description, price, category, image URL).
4. Save.
5. Confirm the product appears in the catalog.

**Screenshot:** Product create form
[Product create form](./screenshots/admin-02-create-product.png)

### Update a product
1. Select edit button beside one of the products.
2. Edit fields.
3. Save changes.
4. Confirm the updated fields appear.

**Screenshot:** Product edit
[Product edit](./screenshots/admin-03-edit-product.png)

### Delete a product
1. Select delete button beside product.
2. Confirm deletion.

**Screenshot:** Product delete
[Product delete](./screenshots/admin-04-delete-product.png)

## 2) Update Order Status

**Goal:** Change an order’s status (e.g., Pending → Shipped).

1. Navigate to bottom of the Admin section.
2. Select an order.
3. Update the status.
4. Save.
5. Confirm the status updates on the admin list (and user order history if visible).
